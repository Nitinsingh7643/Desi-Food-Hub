## üí° Tips for Great Screenshots

- Use **high resolution** (1920x1080 or higher)
- Capture on a **clean browser** (close unnecessary tabs)
- Use **realistic data** (not lorem ipsum)
- Show **key features** prominently
- Consider using **dark mode** for modern look
- Add some **sample orders/products** for context
